# ITS Location Positional Quality

Reusable concepts for describing accuracy, confidence, and uncertainty of a location representation.

This pattern consists of the following classes:

- [Altitude Confidence](AltitudeConfidence.md)
- [Position Accuracy](PositionAccuracy.md)
- [Position Confidence Ellipse](PositionConfidenceEllipse.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](quality-pattern.ttl) and the SHACL [restrictions](quality-shacl.ttl).
