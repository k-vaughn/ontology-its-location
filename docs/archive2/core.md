# ITS Location Referencing Core

Core abstractions for location references and collections used across multiple location-referencing methodologies.

This pattern consists of the following classes:

- [Indexed Location](IndexedLocation.md)
- [Itinerary](Itinerary.md)
- [Location](Location.md)
- [Location Group](LocationGroup.md)
- [Location Reference](LocationReference.md)

The formal definition of this pattern is available in [TURTLE Syntax](core.ttl).
