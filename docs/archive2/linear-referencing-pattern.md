# ITS Linear Referencing (LRS) Location Referencing

Methodology module for locations referenced along a transport network element using measures (linear referencing).

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)
- [https://w3id.org/itsdata/location/v1/NetworkLocationPattern](https://w3id.org/itsdata/location/v1/NetworkLocationPattern)

This pattern consists of the following classes:

- [Linear Location](LinearLocation.md)
- [Measure](Measure.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](linear-referencing-pattern.ttl) and the SHACL [restrictions](linear-referencing-shacl.ttl).
