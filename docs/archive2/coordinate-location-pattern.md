# ITS Coordinate-based Location Referencing

Methodology module for locations referenced using coordinates and/or point geometry.

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)
- [https://w3id.org/itsdata/location/v1/LocationGeometryPattern](https://w3id.org/itsdata/location/v1/LocationGeometryPattern)
- [https://w3id.org/itsdata/location/v1/LocationQualityPattern](https://w3id.org/itsdata/location/v1/LocationQualityPattern)

This pattern consists of the following classes:

- [Coordinate Location](CoordinateLocation.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](coordinate-location-pattern.ttl) and the SHACL [restrictions](coordinate-location-shacl.ttl).
