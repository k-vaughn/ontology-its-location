# ITS Offset-based Location Referencing

Methodology module for locations referenced by offsets from a referent (point or segment) using a distance (and optional direction).

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)

This pattern consists of the following classes:

- [Offset Distance](OffsetDistance.md)
- [Offset Location](OffsetLocation.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](offset-location-pattern.ttl) and the SHACL [restrictions](offset-location-shacl.ttl).
