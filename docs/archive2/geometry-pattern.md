# ITS Location Geometry

Reusable coordinate and geometry primitives for ITS location referencing, aligned to GeoSPARQL.

This pattern consists of the following classes:

- [Point Coordinates](PointCoordinates.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](geometry-pattern.ttl) and the SHACL [restrictions](geometry-shacl.ttl).
