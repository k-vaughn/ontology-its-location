# ITS External Identifier Location Referencing

Methodology module for locations referenced by identifiers in external location referencing systems.

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)

This pattern consists of the following classes:

- [Code](Code.md)
- [Code List](CodeList.md)
- [Location Code](LocationCode.md)
- [Location Code List](LocationCodeList.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](external-id-location-pattern.ttl) and the SHACL [restrictions](external-id-location-shacl.ttl).
