# ITS Address-based Location Referencing

Methodology module for locations referenced using civic/postal addressing structures.

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)

This pattern consists of the following classes:

- [Address Location](AddressLocation.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](address-location-pattern.ttl) and the SHACL [restrictions](address-location-shacl.ttl).
