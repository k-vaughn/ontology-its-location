# ITS Network-anchored Location Referencing

Methodology module for locations referenced relative to transport network elements (using City Data Model Part 3 transport primitives).

This pattern imports the following files:

- [https://w3id.org/itsdata/location/v1/LocationCorePattern](https://w3id.org/itsdata/location/v1/LocationCorePattern)

This pattern consists of the following classes:

- [Network Location](NetworkLocation.md)

The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](network-location-pattern.ttl) and the SHACL [restrictions](network-location-shacl.ttl).
